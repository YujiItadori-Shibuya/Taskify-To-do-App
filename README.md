# Taskify-To-do-App
A modern, minimalist task management web application designed to help users organize their daily lives and boost productivity.

Highlight  on what makes Taskify useful:
User Authentication: Secure Sign-In/Sign-Up and Google Authentication for quick access.
Task Categorization: Organize tasks into preset categories like Personal, Work, Shopping, Health, and Education.
Task Management: Easily add, edit, or delete tasks.
Dynamic Filtering: View tasks based on their status: All, Active, or Completed.
Real-time Updates: Stay organized with a responsive UI that updates as you manage your tasks.
Persistence: Tasks are saved to your account so you can access them from any device.
