Taskify is a modern, minimalist task management web application designed to help users organize their daily lives and boost productivity. Built with a focus on clean UI and seamless user experience, it allows for real-time task tracking and categorization.

**Live Demo:** [https://taskify-fa561.web.app](https://taskify-fa561.web.app)

---

## ‚ú® Features

- **User Authentication:** Secure Sign-In/Sign-Up and Google Auth integration via Firebase.
- **Task Categorization:** Organize tasks into specialized categories: **Personal, Work, Shopping, Health, and Education**.
- **Dynamic Filtering:** Quickly view tasks based on status: **All, Active, or Completed**.
- **Real-time Persistence:** Tasks are synced to your account, ensuring data is available across devices.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
